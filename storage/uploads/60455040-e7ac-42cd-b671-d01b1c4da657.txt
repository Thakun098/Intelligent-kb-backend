อาการ (Symptom): ลูกค้ากว่า 500 รายติดต่อเข้ามาทาง Live Chat โวยวายว่ายังไม่ได้รับเงินคืนหลังจากกดยกเลิกออเดอร์ในแคมเปญ 11.11 ผ่านไปแล้ว 7 วัน สถานะในแอปยังขึ้นว่า "กำลังดำเนินการ"

สาเหตุหลัก (Root Cause): ระบบ Payment Gateway ของพาร์ทเนอร์ (Bank X) มีการจำกัด Rate Limit การยิง API Refund ไว้ที่ 100 transactions/minute ทำให้ระบบของเราที่ยิงคำสั่งคืนเงินพร้อมกันหลักหมื่นรายการถูก reject และค้างอยู่ในคิวโดยไม่มีระบบ Retry อัตโนมัติ

วิธีแก้ไข (Resolution):

ฝ่าย CS ให้แจ้งลูกค้าว่า "ระบบธนาคารกำลังประมวลผลยอดเงินคืนจำนวนมาก คาดว่าจะได้รับเงินภายใน 3-5 วันทำการ" พร้อมเสนอ Vouncher ส่วนลด 50 บาทเป็นการขอโทษ
ฝ่าย Tech ทำการเคลียร์คิวแบบ Manual โดยแบ่งยิง API ครั้งละ 50 รายการต่อนาทีจนครบ
การป้องกัน (Prevention): เพิ่มระบบ Exponential Backoff และ Retry Queue สำหรับ API Refund และเจรจากับ Bank X เพื่อขอขยาย Rate Limit ในช่วง Mega Campaign
